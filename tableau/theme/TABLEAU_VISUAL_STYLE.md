# Tableau Visual Style

Use the approved portfolio visual language. Do not copy any publication brand or template.

## Palette

| name | hex | use |
|---|---|---|
| Ink | `#1F2933` | text and axes |
| Warm background | `#F7F5F0` | dashboard background |
| Accent red | `#B23A48` | poverty and warnings |
| Research blue | `#2F5F8F` | regional context |
| Technology teal | `#1F7A8C` | social protection and dashboard actions |
| Warm orange | `#E58F2A` | informality and labor-market emphasis |
| Muted green | `#5E7D68` | coverage or lower-risk states |
| Light gray | `#D9DDE3` | separators and missing data |
| Medium gray | `#7A828E` | secondary labels |

## Rules

- Use a light background and generous white space.
- Use maximum three relevant colors per dashboard.
- Use finding-based titles, not generic chart names.
- Keep methodological notes visible on every dashboard.
- Do not use 3D charts, gauges, decorative icons or excessive KPI cards.
- Use maps only when geographic comparison adds value.
- Clearly distinguish countries with missing data.
- Format percentages consistently with one decimal unless risk communication requires two.
- Keep the style original and analytically restrained.

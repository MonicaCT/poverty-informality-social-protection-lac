# Recruiter Guide

This project can be reviewed in under five minutes as evidence of Development Data Analyst, Policy Data Analyst, Social Protection Analyst, Research Data Analyst and Monitoring and Evaluation Analyst skills.

## Policy Problem

The repository turns fragmented poverty, labor informality and social-protection indicators into a harmonized country-year analytical panel for Latin America and the Caribbean.

## Analytical Challenge

The challenge is not only visualization. It includes source integration, country-year harmonization, missingness assessment, comparability caveats, public dashboard communication and responsible interpretation of descriptive cross-country indicators.

## Tools

- Python for panel construction, descriptive analysis and dashboard generation.
- Quarto and Plotly for the interactive dashboard.
- GitHub Pages for public communication.
- Markdown documentation for reproducibility, lineage, validation and portfolio review.
- SQL assets for visible analytical modeling and future Tableau preparation.
- Automated repository tests for integrity and publication readiness.

## Workflow

1. Inventory source outputs.
2. Build a harmonized country-year panel.
3. Validate duplicates, impossible values, missingness and coverage.
4. Generate descriptive figures and executive tables.
5. Build a public interactive dashboard.
6. Document limitations, lineage and reproducibility.

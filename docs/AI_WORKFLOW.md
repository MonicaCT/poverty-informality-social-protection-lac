# AI-Assisted Workflow Documentation

AI assistance was used to accelerate reproducible coding, inventory design, documentation drafting, and workflow organization. The substantive research decisions were constrained by the data inventory and by standard applied econometric practice.

## Important AI-Assisted Decisions

- Use the completed data inventory before selecting the empirical strategy.
- Use harmonized country-year outputs as the first analytical backbone instead of forcing all raw survey files into one immediate pipeline.
- Keep Bolivia household survey microdata as a validation and extension layer.
- Treat two-way fixed effects as the preferred baseline and dynamic GMM as robustness due to sample size and instrument concerns.
- Export every major table and model in open formats and publication formats.

## Reproducibility Guardrails

- All generated outputs are script-based.
- Raw local data are never modified.
- Source paths are documented.
- Every derived variable is listed in the data dictionary.
- Model limitations are written into the interpretation files.

# Accessibility Checklist

- [ ] Use light background with high-contrast text.
- [ ] Keep font sizes readable on laptop screens.
- [ ] Avoid red-green-only distinctions.
- [ ] Label missing data explicitly.
- [ ] Use no more than three relevant colors per dashboard.
- [ ] Add direct labels where feasible.
- [ ] Keep chart titles descriptive.
- [ ] Provide methodology notes on every dashboard.
- [ ] Avoid cluttered legends.
- [ ] Ensure dashboard navigation can be understood without color alone.
